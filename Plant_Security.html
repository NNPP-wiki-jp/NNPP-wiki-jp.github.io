Plant Security

"DoE Security Forces (SECFOR) is responsible for checkpoint operations and interior security of the Naramo Power Plant."
— DoE Security Forces (SECFOR) は、ナラモ発電所における検問所業務および施設内部の警備を担当している。

概要

Plant Securityは、NNPPにおける初期役割の一つである。この役割を進めるめることで、30,000 XPでTactical Response Team、50,000 XPでGround Service Personnelが解放される。Senior Guard以上になると、民間人の施設立ち入りを許可する権限が付与される。

任務内容

主な任務は、West Noobianによる原子炉破壊を阻止することである。マップ上の指定ポイント（チェックポイント、制御室、発電機、冷却室）で待機することで、30秒ごとに5 XPを獲得できる。

  • 敵の識別： 暗い服、赤いネームタグ、ライフルの携行が特徴。撃破時に30 XPを獲得。
  • 民間人対応： 無許可の民間人を排除・制圧する（XP獲得はなし）。
